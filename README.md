## Description
A tech news website where users can post, upvote, and comment on links to news articles. The app was refactored from Node.js and Handlebars.js to use Python and the Python Flask framework.
## Technologies
- python
- flask
- gunicorn
- SQLAlchemy
- PyMySQL
